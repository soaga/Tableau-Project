# Airbnb USA Market Analysis | Tableau

## Project Overview
This project analyzes Airbnb listings across the United States to understand pricing patterns, location trends, and revenue potential.

## Tools Used
- Tableau
- Data Visualization
- Data Cleaning

## Objectives
- Analyze Airbnb pricing trends across cities
- Identify high-revenue locations
- Visualize listing distribution across the USA

## Key Insights
- Certain cities generate significantly higher Airbnb revenue
- Location strongly influences pricing
- Entire homes/apartments dominate listings

## Dashboard Features
- Interactive filters by location and room type
- Price distribution maps
- Revenue analysis charts
